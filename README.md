# 📊 Retail Sales SQL Analysis Project

## 📌 Project Overview
This project is a SQL-based retail sales analysis that focuses on understanding business performance using real-world transactional data. The main objective is to clean the dataset, explore key patterns, and generate meaningful insights that support better business decision-making.

The dataset contains retail transaction details such as transaction ID, sale date and time, customer information, product categories, quantity sold, and total sales amount.

In this project, I performed data cleaning by handling missing values and fixing column name issues caused during data import. After cleaning the dataset, I conducted exploratory data analysis (EDA) to understand data structure and sales distribution.

Finally, I used SQL queries to analyze key business problems such as identifying top customers, analyzing category-wise sales, finding monthly trends, and understanding sales patterns across different time shifts.

This project demonstrates practical SQL skills and the ability to convert raw data into meaningful business insights.
